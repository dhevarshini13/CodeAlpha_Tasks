This repository contains the projects I built during my **Python Programming Internship at CodeAlpha**.

## Projects I Worked On

###  Chatbot

This is a simple rule-based chatbot that can interact with users through predefined responses.
It responds to basic inputs like greetings and common questions.

*What I learned:*
I understood how to use conditional statements (`if-elif`), loops, and functions to simulate conversations.

### Hangman Game

A fun text-based game where the user guesses a hidden word one letter at a time.
The player has limited chances to guess the word correctly.

*What I learned:*
I practiced working with strings, lists, loops, and the `random` module while building a small game.

###  Stock Portfolio Tracker

This project helps calculate the total value of stocks based on user input and predefined prices.

*What I learned:*
I learned how to use dictionaries, take user input, and perform basic calculations. It also gave me an idea of how simple programs can be used in finance-related tasks.
